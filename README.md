# PinkBlue

## Animações Sprite Kit
Utilização de Sprites public domain para construir uma animação em que o personagem se movimenta na tela onde o usuário clicar. Além disso o usuário pode trocar de cena clicando num botão indicando o avatar de um dos personagens.
